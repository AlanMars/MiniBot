Detailed Description

Any information about the robot's current status should be accessed through the superstructure.

This class can be thought of as an interface for getting high-level data about the robot as well as high-level methods for controlling components of the robot that interact with eachother. (ex. An elevator and arm could be both controlled with a setIntakeKinematicPosition(x,y); )
